# Creality slicer profiles

Save exported Creality print/slicer configuration profiles here so good prints
can be reproduced exactly.

## Exporting from Creality Print

- Open **Creality Print**.
- Set up your printer + print settings the way you want to keep them.
- In the print-settings panel, use the profile menu → **Export** (or
  right-click the profile → Export). This produces a config file (e.g.
  `.ini` / `.curaprofile` / `.creality_printer` depending on version).
- Drop the exported file in this folder and give it a descriptive name, e.g.
  `PLA_0.2mm_thistle.ini`.

Reference the profile name from a print's `print-notes.md` so it's clear which
profile produced which physical print.
